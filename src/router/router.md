##### Router 构建

**1、文件**

| 文件             | 描述                                                                        |
| ---------------- | --------------------------------------------------------------------------- |
| **basicRoutes/** | 静态二级目录，统一 BasicLayout 布局，不可重复                               |
| **dynaRoutes/**  | 动态二级目录，export 布局组件及视图，自定义布局，文件名为一级 path,不可重复 |
| **index**        | 动态引入，定义基础路由，需 export default方式                              |
